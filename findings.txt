For any m×n transportation problem, every IBFS method must make 
exactly m+n−1 allocations to get a basic feasible solution. For 
a 5×6 problem that is always exactly 10 iterations — no method can 
do it in fewer. So RAM and VAM will always tie on iteration count 
for the same problem size. Asking for fewer RAM iterations than VAM 
is mathematically impossible for the same-sized problem.
